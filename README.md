# Analisis Tingkat Kemiskinan di Indonesia
## Deskripsi Proyek
Proyek ini berfokus pada analisis pembangunan regional di Indonesia dengan memanfaatkan data terbuka dari Kaggle. Analisis dilakukan untuk memahami disparitas antarwilayah berdasarkan empat dimensi utama, yaitu ekonomi, kemiskinan, pasar tenaga kerja, dan infrastruktur dasar.

Seluruh hasil analisis divisualisasikan menggunakan Power BI dengan pendekatan visualisasi interaktif dan geospasial, termasuk penggunaan Shape Map dan Squillion Map untuk menampilkan pola dan kesenjangan pembangunan di tingkat Provinsi serta Kabupaten/Kota.
